# Sohaib Akhlaq — Portfolio (Static)

This is a **high-UI, responsive portfolio website** built as a **static site** (no npm/build required).  
All your content is stored in **`data.js`** so you can update it easily.

## Run locally

- **Fastest**: open `index.html` in your browser.
- **Recommended** (to avoid browser file restrictions):
  - If you have Python:
    - `python -m http.server 5500`
    - Then open `http://localhost:5500`

## Edit your data

- Update everything inside **`data.js`**:
  - About text
  - Tech stack icons
  - Education
  - Achievements
  - GitHub username (projects auto-load from GitHub)

## Deploy (GitHub Pages)

1. Create a repo (example: `sohaib-portfolio`)
2. Upload the contents of this folder (`index.html`, `styles.css`, `script.js`, `data.js`, `assets/`)
3. Enable GitHub Pages for the repo (Deploy from branch → root)

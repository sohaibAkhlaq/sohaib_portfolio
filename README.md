Professional portfolio engineered for an AI Engineer and Full-Stack Developer. This repository highlights the transition towards building scalable LLM-powered automation systems, agentic workflows, and full-stack intelligent platforms.

## 🚀 Featured AI Projects

1. **[Medical Report Generation](https://github.com/sohaibAkhlaq/medical-report-generation)**
   A Zero-Hallucination Multimodal Generative AI System that generates factual radiology and pathology reports. Features rigorous vision-language grounding via BiomedCLIP, contradiction detection, and fully attributed evidence logs.
   *Tech Stack: Python, FastAPI, Docker, React, Groq, Gemini, BiomedCLIP*

2. **[DischargeAI](https://github.com/sohaibAkhlaq/dischargeai)**
   An autonomous clinical agent designed to resolve hospital ward handover bottlenecks. Transforms unstructured notes into highly structured discharge summaries and HL7 FHIR bundles with strict rule-based escalation protocols.
   *Tech Stack: Llama-3.3-70b, HL7 FHIR, FastAPI, JavaScript, Agentic AI*

3. **[Calder AI Automation](https://github.com/sohaibAkhlaq/calder-ai-automation)**
   A production-ready intelligent workflow platform that automates customer support ticket classification, sentiment analysis, and urgency detection.
   *Tech Stack: FastAPI, Sentence Transformers, Groq API, Docker, Python*

## 💻 Running Locally

To preview the portfolio locally, you can use any static server. For example:

Using Python:
```bash
python -m http.server 8080
```

Using Node (npx):
```bash
npx http-server -p 8080
```
Then navigate to `http://localhost:8080` in your web browser.
